== Description ==

This plugin allows you to use these image files
- heic
- heif
- heics
- heifs
- avif
- avifs

You can convert your images for free on https://avif.io/

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Thats it, nothing to configure

== Frequently Asked Questions ==

1. I still can't upload AVIF files and get an error message
It can happen that - depending on your serverstructure - WordPress can't correctly recognize the correct mime type of your AVIF files. In that case you need to install one addition plugin, you can find it here:
https://wordpress.org/plugins/blob-mimes/
